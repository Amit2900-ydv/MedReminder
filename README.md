
  
# 🤖 MedReminder – AI Powered Smart Medication Assistant

MedReminder is an AI-powered smart medication management application designed to improve medicine adherence through intelligent tracking, voice interaction, and chatbot assistance.

---

## 📌 Problem Statement

Medication adherence is a major challenge in healthcare.  
Patients often forget to take medicines on time, leading to health risks and reduced treatment effectiveness. Traditional reminder apps lack intelligence and personalization.

MedReminder aims to solve this problem using Artificial Intelligence.

---

## 🚀 Features

### 🏠 Smart Dashboard
- Displays adherence score
- Shows today’s medicines
- Tracks missed doses
- Generates AI insights

### 🔔 Smart Alerts
- Daily medication reminders
- Missed dose detection
- Status tracking (Taken / Missed / Verified)

### 🤖 AI Chat Assistant
- Interactive chatbot support (Google Gemini)
- Medicine-related queries
- Multi-language support (English, Hindi, Hinglish, Marathi, Gujarati)
- Voice greeting on startup

### 📊 Reports & Analytics
- Adherence percentage calculation
- Daily performance summary
- Taken vs Missed ratio

### 📷 Medicine Scanner
- Scan medicines before intake
- Reduces manual errors

---

## 🧠 Adherence Calculation Logic

Adherence Percentage is calculated as:

(Taken Medicines / Total Scheduled Medicines) × 100

This helps users track consistency and improve medication discipline.

---

## 🎤 Voice Integration

- AI greets user on app startup
- Text-to-Speech functionality
- Language preference support

## 🛠 Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, Motion
